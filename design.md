# Logics
## DSP
 * DSP default color: blue
 * select DSP, color it green
 * pressing program button:
   * Selected DSP colors red
 * pressing reset:
   * disables program
   * waits (configurable)
   * enables reset
   * wait (configurable)
   * disables reset  
   * colors DSP green
 * selecting an already selected DSP, deselects it     
 * selecting a deselected DSP deselects all other DSPs (only one can be active)

## Carriers
 * carriers can be either on (green) or off (red)
 * multiple carriers can be on at the same time
 * carrier state is persisted to disk, and read on startup
